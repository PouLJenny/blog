# 数据库


## 参考书

《数据库系统概论》（第5版） - 作者：王珊、萨师煊
《数据库系统概念》（第6版） - 作者：阿巴拉契亚州立大学霍克斯教授等
《数据库设计与实现（第2版）》 - 作者：叶乃琳、陈建华
《数据库原理及应用（第3版）》 - 作者：何钦铭
《数据库技术及应用》 - 作者：戴林鸿
《数据库原理（第3版）》 - 作者：朱乾坤
《数据库系统概论（第4版）》 - 作者：周立功、贾维斯
以上书籍都是比较经典的数据库教科书，涵盖了数据库的基本概念、原理、应用和实践等方面的内容，适合初学者学习。可以根据自己的需求和兴趣选择一本或多本进行学习。
